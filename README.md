## 🧠 About the Project  

MOJITO is a visually stunning landing page inspired by the Velvet Pour website.  
Built using **React and GSAP**, this project focuses on delivering a premium user experience through smooth animations, modern UI, and responsive design.

It replicates the elegance and aesthetic of a cocktail brand website with interactive elements and clean layouts.

---

## ✨ Features  

- 🎬 Smooth and engaging GSAP animations  
- 🎯 Modern and minimal UI design  
- 📱 Fully responsive across all devices  
- 🍸 Cocktail showcase section  
- 🧾 About section with storytelling  
- 🎨 Artistic visual section  
- 📍 Contact & location section  
- 🌿 Clean and aesthetic layout  

---

## 🛠️ Tech Stack  

- **Frontend:** React + JavaScript  
- **Styling:** Tailwind CSS  
- **Build Tool:** Vite  
- **Animations:** GSAP  
- **Media Processing:** FFmpeg
